# The Long Noon _(working title)_

A cozy, sunny third-person survival/crafting/exploration game with a quietly devastating story buried underneath.

> A world where nobody can die, nobody has family, and nobody remembers why. You craft your way into older and older regions and slowly uncover what the alien took from everyone.

**Status:** Pre-production — design phase.
**Target:** Steam (Windows first; Linux + macOS later).

## Engine
**Unreal Engine 5 (5.4), C++ + Blueprints.** Skeleton on branch `feature/unreal-scaffold`. To get to a first playable build, follow [docs/tech/first-playable-checklist.md](docs/tech/first-playable-checklist.md) (requires UE5 installed).

## Pillars
- **Cute on top, devastating underneath** — the warmth *is* the wound.
- **Regions = story** — gear/crafting gates regions; descending into older regions reveals the buried truth.
- **Non-lethal "combat"** — weapons are reclamation tools against the overgrown, undying Bloom.
- **Rediscovery** — the world forgot; the player remembers.
